## 🔧 Fix: Browser Freeze saat Load 2000+ File

**Masalah:** Browser freeze saat `app.py` dijalankan karena folder `01_pdf_source` dan `02_pdf_target` berisi total ~359 PDF (dan subfolder lainnya). API `/api/files` mengembalikan semua file sekaligus, dan frontend merender semua ke DOM sebagai `<li>` element.

**Solusi:**
1. **Backend (`app.py`):** `get_pdf_list()` sekarang return `{"files": [...], "total": N, "truncated": true/false}` dengan limit default 100 file.
2. **Frontend (`templates/index.html`):** `loadFiles()` menangani struktur baru, hanya render 100 `<li>` max, menampilkan info "… dan X file lainnya (total Y)" jika truncated.

**File terubah:**
- `app.py` — `get_pdf_list()` + `api_files()`
- `templates/index.html` — `loadFiles()`
- `Dashboard.md` — status updated
